Implement a GPU kernel that, for N three-dimensional points stored on the device, fills indices[i] with the index j ≠ i of the point closest to points[i]. Comparing squared Euclidean distance is sufficient— you do not need to compute square-roots.

Implementation Requirements

  • The solve function signature supplied by the harness must remain unchanged.
  • Do not allocate host-side memory inside solve.
  • No external libraries (Thrust, cuBLAS, etc.) may be used.
  • Write the output directly into the provided indices array.

Example 1

Input:   points  = [(0,0,0), (1,0,0), (5,5,5)]
         indices = [-1, -1, -1]
         N       = 3

Output:  indices = [1, 0, 1]   # 0⇆1 are nearest, 2 is closest to 1

Constraints

  • 1 ≤ N ≤ 100 000
  • Coordinates are 32-bit floats in the range [-1000, 1000]
