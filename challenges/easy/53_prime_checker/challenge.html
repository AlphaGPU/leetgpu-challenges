Given a 32-bit signed integer n, implement a program that stores 1 in the output if the integer is prime, and 0 otherwise. A prime number is an integer greater than 1 that has no positive divisors other than 1 and itself.

Implementation Requirements

  • Use only native features (external libraries are not permitted)
  • The solve function signature must remain unchanged
  • The final result must be stored in the output variable

Example 1:

Input:  n = 2
Output: output = 1

Example 2:

Input:  n = 9
Output: output = 0

Example 3:

Input:  n = 113
Output: output = 1

Constraints

  • 1 ≤ n ≤ 100,000,000
