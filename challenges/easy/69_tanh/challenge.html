Write a GPU kernel that applies the hyperbolic tangent (tanh) activation function element-wise to a 1D array of 32-bit floating point numbers. Given an input array X of length N, compute Y[i] = tanh(X[i]) for all i. The tanh function maps any real number to the range (-1, 1).

Implementation Requirements

  • Use only native features (external libraries are not permitted)
  • The solve function signature must remain unchanged
  • The final result must be stored in array Y

Example:

Input:  X = [0.0, 1.0, -1.0, 2.0], N = 4
Output: Y = [0.0, 0.76159, -0.76159, 0.96403]

Constraints

  • 1 ≤ N ≤ 100,000,000
  • Input values are 32-bit floats in the range [-1000, 1000]
  • Output values are 32-bit floats in the range (-1, 1)
  • Performance is measured with N = 50,000,000
