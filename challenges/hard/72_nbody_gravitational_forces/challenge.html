Given N particles in three-dimensional space, compute the gravitational acceleration on each particle due to all other particles. Each particle i has a 3D position (x, y, z) and a scalar mass m. All inputs and outputs are 32-bit floating point numbers.

The gravitational acceleration on particle i is the vector sum of contributions from every other particle j ≠ i: 

acceleration[i] = ∑j ≠ i  m[j] × (pos[j] - pos[i]) / (||pos[j] - pos[i]||² + ε)3/2

where ε = 1e-9 is a softening constant that prevents division by zero when two particles occupy the same position. Positions are stored in row-major layout: positions[i*3+0], positions[i*3+1], positions[i*3+2] are the x, y, z coordinates of particle i. The output accelerations array uses the same layout.

Implementation Requirements

  • Use only native features (external libraries are not permitted)
  • The solve function signature must remain unchanged
  • The final result must be stored in the accelerations array

Example

Input:
  N = 3
  positions = [[0, 0, 0],
               [1, 0, 0],
               [0, 1, 0]]
  masses    = [1.0, 1.0, 1.0]

Output:
  accelerations ≈ [[ 1.0,     1.0,    0.0],
                   [-1.354,  0.354,  0.0],
                   [ 0.354, -1.354,  0.0]]

Explanation:
  Particle 0 is pulled toward particle 1 (delta=[1,0,0], r=1) and particle 2 (delta=[0,1,0], r=1).
  Acceleration from particle 1: 1.0 * [1,0,0] / 1 = [1, 0, 0]
  Acceleration from particle 2: 1.0 * [0,1,0] / 1 = [0, 1, 0]
  Total: [1, 1, 0]

Constraints

  • 1 ≤ N ≤ 65,536
  • Positions are 32-bit floats
  • Masses are non-negative 32-bit floats
  • Performance is measured with N = 16,384
